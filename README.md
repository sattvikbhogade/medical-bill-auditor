# Medical Bill Auditor

A production-ready monorepo scaffold for an AI-powered medical bill auditing platform.

## Repository Structure

- frontend/ — React, Vite, TypeScript, and Bootstrap-based client
- backend/ — FastAPI Python service
- docs/ — project documentation and architecture notes
- datasets/ — reference and benchmark datasets
- docker/ — container build files
- scripts/ — automation and maintenance scripts

## Included Files

- Monorepo directories and application placeholder folders
- Environment example files for frontend and backend
- Docker Compose configuration for frontend, backend, and PostgreSQL
- Base .gitignore for Node, Python, Docker, and local environment files

## Notes

This scaffold intentionally includes only structure and configuration. It does not install dependencies or implement application logic.
